## ✨ Project Overview
PulseBoard AI is a mobile and web application designed to centralize and track Artificial Intelligence trends. It delivers an immersive, ultra-modern user experience inspired by Apple's **Glassmorphism** design standards.

## 🚀 Core Features
- **Liquid Glass Dashboard**: A deep-blue interface utilizing custom gradients and light-refraction techniques to simulate frosted glass.
- **Activity Streak (Gamification)**: A dynamic progress bar system to track 7-day user engagement and learning consistency.
- **Pulse Cards System**: Interactive, sleek cards to manage and categorize AI learning subjects.
- **Real-time System Monitoring**: Visual indicators for AI system status (Optimal/Down).
- **Floating Glass Navigation**: A custom-built bottom navigation bar with translucent depth.

## 🛠️ Tech Stack
- **Framework**: Expo (React Native)
- **Navigation**: Expo Router (File-based routing)
- **UI Architecture**: 
  - `expo-linear-gradient` for advanced glass simulation.
  - `Lucide / FontAwesome` for crisp iconography.
  - Native `Animated` API for fluid transitions.
